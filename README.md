# inhabit-module-tests [![npm version](https://badge.fury.io/js/inhabit-modules-tests.svg)](https://badge.fury.io/js/inhabit-modules-tests)
A test suite for any InHabit Module

## Installation
```bash
npm install --save-dev inhabit-modules-tests
```

## Usage
```javascript
let tests = require('inhabit-modules-tests');

tests(process.cwd());
```
